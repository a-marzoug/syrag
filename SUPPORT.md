# Support

SyRAG is an open-source project maintained on GitHub.

## Questions

For usage questions, start with:

- [README](README.md)
- [Documentation](https://a-marzoug.github.io/syrag/)
- [Cookbook](docs/cookbook/index.md)

If the docs do not answer the question, open a GitHub issue with enough context to reproduce or understand the workflow.

## Bugs

Open a bug report and include:

- SyRAG version
- Python version
- installed extras
- minimal reproduction
- expected behavior
- actual behavior

## Security

Do not open public issues for suspected vulnerabilities. Follow the [security policy](SECURITY.md).
